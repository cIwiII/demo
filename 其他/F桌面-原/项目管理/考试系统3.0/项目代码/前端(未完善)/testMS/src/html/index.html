蜗牛创想7周年 | 探寻职业教育的本质

今天这篇文章为蜗牛创想CEO邓强老师对内的全员邮件，在公司成立7周年这个特殊的日子，邓老师向大家分享了蜗牛创想近年来在探寻职业教育本质上的一些思考和体会。


各位同事，大家好！
